# Фабрика гипотез

Учебный RAG-прототип для генерации проверяемых научно-технических гипотез.

Система использует демо-корпус, Qdrant retrieval, локальный граф
знаний (`data/knowledge_graph/graph.json`) и DeepSeek API, чтобы вернуть
гипотезы с источниками, механизмом влияния, рисками, ожидаемым KPI и планом
лабораторной проверки. Опционально подключите папку `Задача 1` из архива
конкурса (не входит в Git) и выполните `python -m src.ingest --recreate`.
Цитаты проверяются на соответствие retrieved context; отчёт можно скачать в MD/DOCX.

Перед запуском чата выполните:

```bash
docker compose up -d qdrant
python -m src.ingest --recreate
```

Или запустите всё одной командой: `docker compose up --build`.

`src.ingest` одновременно обновляет Qdrant и пересобирает
`data/knowledge_graph/graph.json`.

Для повторного вывода последнего отчета напишите в чат:

```text
/export
```
