# Agent Notes

This project is a local-first MVP for the hackathon task "Фабрика гипотез".

Development rules:

- Default mode must work without paid APIs, GPUs, external databases, or network access.
- Treat local evidence from `../Задача 1` as authoritative for demo outputs.
- Generated hypotheses must include evidence, causal chain, risks, validation plan, score breakdown, and warnings.
- Do not let an LLM-only answer bypass validation or scoring.
- Keep all exports under `data/demo_outputs/`.
